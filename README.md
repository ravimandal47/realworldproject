
## Real World Project ##

  Project Title: Fake Product Identification System using Blockchain

- Tech Stack
	- Frontend: ReactJS
	- Backend: Node.JS 
	- Database: MongoDB
	- Mobile Application : Flutter 
	- Blockchain : Ethereum and Blockchain 


Research Paper: https://ieeexplore.ieee.org/stamp/stamp.jsp?arnumber=8985337

    cd rwp-api
